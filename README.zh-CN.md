## ✨ 关于这个产品

HappyPartners 是一款 macOS 原生桌面应用程序，让你在同一个画面上同时操作多个 AI 平台，像主持人带领一场多人聊天室。

支持 **16 个 AI 平台**（画面上最多同时显示 **4 个**）。一次发消息、同步比对、跨 AI 拖曳转传，所有对话本机保存。

不是浏览器扩展、不是 Web 平台、不是 API 代理。你用自己的 AI 订阅登入，资料只存在你的电脑。

## 🎯 适合谁用

不是给偶尔用 AI 的人。是给每天真的把多个 AI 当工作工具的重度使用者：

- **创作者、设计师、工程师、产品经理**
- **研究者、翻译、文案、顾问**
- 任何需要在不同 AI 之间来回比对、接力、整理的人

## 🧠 核心特色

### 一个工作区，整个 AI 团队
最多同时显示 4 个 AI 窗口，共用输入框。一次发给全部，或用 `@` 指定对象。

### 跨 AI 无缝接力
按住 `Alt` 拖曳把任一 AI 的回复转给另一个 AI，或广播给画面上其他所有 AI。内容搬运由 app 完成。

### 你的账号．你的数据．你的电脑
用你原本的 AI 订阅登入，不需 API Key、不抽成、不碰对话内容。所有数据只存在你的 Mac。

### 为日常工作而设计
对话可归档成项目，每个项目有独立的 AI 组合与版面。草稿自动保存，中文友善的全文搜索（FTS5 trigram），回查上次的脉络很快。

## 🌐 支持平台

**16 个平台**可用。界面支持繁體中文、简体中文、English — 切换界面语言时平台名称跟着换。

ChatGPT · Claude · Gemini · Grok · Perplexity · Copilot · AI Studio · Mistral · DeepSeek · Kimi · Qwen · Yuanbao · Doubao · MiMo · Z.ai · ChatGLM

## ⚖️ 和其他工具的差异

HappyPartners 不提供 AI 模型，也不代理 API。你使用的是各平台原本的官方服务与你自己的账号 — 我们只是把操作方式整合到同一个工作空间中。

- ✅ **没有额外的 token 计费** — 你原本就要付的 AI 订阅不变
- ✅ **不需要多一层 API 平台**要维护
- ✅ **没有第三方**看得到你的对话内容

## 💬 反馈与支持

- **[GitHub Issues](https://github.com/happypartners-app/HappyPartners/issues)** — Bug 反馈、功能建议、讨论
