========================================================
  HappyPartners
========================================================

你的 AI 夥伴團隊，終於在同一個地方了。
少一點搬運，多一點協作。

官網：https://happypartners.app


═══════════════════════════════════════════════════════════
  關於這個產品
═══════════════════════════════════════════════════════════

HappyPartners 是一款 macOS 原生桌面應用程式，讓你在同一個畫面上
同時操作多個 AI 平台，像主持人帶領一場多人聊天室。

支援 16 個 AI 平台（畫面上最多同時顯示 4 個）。一次送訊息、同步
比對、跨 AI 拖曳轉傳，所有對話本機保存。

不是瀏覽器擴充、不是 Web 平台、不是 API 代理。你用自己的 AI 訂
閱登入，資料只存在你的電腦。


═══════════════════════════════════════════════════════════
  適合誰用
═══════════════════════════════════════════════════════════

不是給偶爾用 AI 的人。是給每天真的把多個 AI 當工作工具的重度使
用者——創作者、設計師、工程師、產品經理、研究者、翻譯、文案、
顧問……任何需要在不同 AI 之間來回比對、接力、整理的人。


═══════════════════════════════════════════════════════════
  核心特色
═══════════════════════════════════════════════════════════

一個工作區，整個 AI 團隊
  最多同時顯示 4 個 AI 視窗，共用輸入框。一次送給全部,或用 @
  指定對象。

跨 AI 無縫接力
  Alt + 拖曳把任一 AI 的回覆轉給另一個 AI,或廣播給畫面上其他
  所有 AI。內容搬運由 app 完成。

你的帳號．你的資料．你的電腦
  用你原本的 AI 訂閱登入,不需 API Key、不抽成、不碰對話內容。
  所有資料只存在你的 Mac。

為日常工作而設計
  對話可歸檔成專案,每個專案有獨立的 AI 組合與版面。草稿自動保
  存,中文友善的全文搜尋,回查上次的脈絡很快。


═══════════════════════════════════════════════════════════
  支援平台
═══════════════════════════════════════════════════════════

ChatGPT · Claude · Gemini · Grok · Perplexity · Copilot · AI Studio ·
Mistral · DeepSeek · Kimi · Qwen · Yuanbao · Doubao · MiMo · Z.ai ·
ChatGLM

三語介面(繁體中文、簡體中文、English),切換介面語言時平台名稱
跟著換。


═══════════════════════════════════════════════════════════
  和其他工具的差異
═══════════════════════════════════════════════════════════

HappyPartners 不提供 AI 模型、也不代理 API。你使用的是各平台原
本的官方服務與你自己的帳號,只是把操作方式整合到同一個工作空
間中。

  · 沒有額外的 token 計費
  · 不需要把工作流建立在另一層 API 平台之上
  · 不用擔心對話內容經過第三方

你原本就要付的 AI 訂閱不變,HappyPartners 改變的只是「同時操作
多個 AI」這件事的體驗。
